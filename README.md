### AI CV Optimizer & Architect
A full-stack platform using Gemini 3.0 to tailor resumes to job descriptions in real-time.

### Key Features:
ATS Scoring: Instant match percentage and keyword gap analysis.
Live Editor: A "paper-style" preview to edit AI improvements.
Docx Export: One-click download of the optimized document.

### Tech Stack
Backend: FastAPI (Python), LangChain, python-docx.
Frontend: React (Vite), Tailwind-inspired CSS.
AI: Google Gemini 3.0 API.

### Setup
Backend: pip install -r requirements.txt && python main.py
Frontend: npm install && npm run dev
